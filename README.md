# dyndns-hetzner
Simple DNS updater for custom / homemade DynDNS with Hetzner Console - supports only one domain (there are no plans for more)

Works with Ubuntu 24.04 (should work in most current linux derivates). 

Prequisites:
- Hetzner Account
- Domain (which is managed with the Hetzner Console)
- Token for Hetzner-API-Calls

Install:
1. checkout repository
2. chmod +x install.sh
3. configure /etc/dyndns-hetzner/dyndns-hetzner.conf

Deinstall:
1. ¯\_(ツ)_/¯
2. web search "how to remove a custom service" :)
